# 연산자 우선순위

| 우선순위 | 연산자         | 연산                  |
|:----:|:-----------:|:-------------------:|
| 높음   | **          | 지수                  |
|      | -           | 음수 부호               |
|      | *, /, //, % | 곱셈,나눗셈, 정수 나눗셈, 나머지 |
| 낮음   | +, -        | 덧셈, 뺄셈              |

(ex) 
-2 ** 4 = -16임  → **음수는 괄호로 묶어서 사용하는 습관 들이기** → (-2) ** 4 = 16
